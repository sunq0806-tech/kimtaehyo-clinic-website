# 예약 폼 연동 가이드 (구글 폼 + 구글 시트 + 이메일 알림)

현재 홈페이지의 예약 요청 폼은 **구글 폼(Google Forms)으로 바로 전송**되도록 이미 연결되어 있습니다.
방문자가 폼을 제출하면 → 구글 폼 응답에 쌓이고 → 연결된 구글 시트에 자동 저장됩니다.
아래 순서대로 "응답 확인 방법"과 "이메일 알림 켜기"만 설정하시면 됩니다.

## 1단계. 응답이 쌓이는 구글 폼 열기

1. [forms.google.com](https://forms.google.com) 접속 (폼을 만들 때 사용한 구글 계정으로 로그인)
2. 예약 요청용 폼 열기 → 상단 **응답** 탭에서 접수된 예약 요청을 확인할 수 있습니다.

## 2단계. 구글 시트로 자동 저장 연결

1. 폼의 **응답** 탭에서 오른쪽 위 **시트 아이콘(초록색)** 클릭
2. "새 스프레드시트 만들기" 선택 → 이름을 `김태효탑내과_예약요청`으로 지정
3. 이후 모든 예약 요청이 이 시트에 실시간으로 한 줄씩 추가됩니다.

## 3단계. 새 예약이 오면 이메일 알림 받기

1. 폼의 **응답** 탭 → 오른쪽 위 **⋮ (더보기)** 클릭
2. **"새 응답에 대한 이메일 알림 받기"** 체크
3. 알림은 폼 소유자 계정의 이메일로 발송됩니다. 원장님/실장님이 실제 확인하는 계정으로 폼 소유권을 공유해두세요.
   (폼 오른쪽 위 ⋮ → 공동작업자 추가)

## 4단계. 테스트

1. 홈페이지( index.html )를 열고 예약 폼을 실제로 한 번 제출
2. 구글 폼 응답 탭 + 연결된 시트에 새 행이 추가되는지 확인
3. 알림 이메일이 도착하는지 확인

## 폼 항목을 바꾸고 싶을 때 (주의)

홈페이지 코드는 구글 폼의 각 질문에 붙는 고유번호(entry ID)로 값을 전송합니다.
`website/index.html` 맨 아래 스크립트의 이 부분입니다:

```js
const FORM_ACTION = 'https://docs.google.com/forms/d/e/…/formResponse';
const FORM_FIELDS = {
  name: 'entry.631252392',
  tel:  'entry.143630468',
  ...
};
```

- 구글 폼에서 **질문을 추가/삭제하면 entry 번호가 바뀔 수 있으므로**, 폼을 수정한 뒤에는 반드시 홈페이지에서 테스트 제출을 해서 시트에 값이 정상으로 들어오는지 확인하세요.
- entry 번호 확인법: 구글 폼 미리보기 화면에서 마우스 오른쪽 → "페이지 소스 보기" → `entry.` 검색.

## (부록) Apps Script 방식으로 바꾸고 싶다면

구글 폼 대신 Apps Script 웹앱으로 받는 방법도 있습니다. `apps-script-code.gs` 파일이 그 용도입니다.

1. [sheets.new](https://sheets.new)에서 새 시트 생성 → **확장 프로그램 → Apps Script**
2. `apps-script-code.gs` 내용을 붙여넣고, `NOTIFY_EMAIL`을 실제 이메일로 수정 후 저장
3. **배포 → 새 배포 → 웹 앱** (실행: 나 / 액세스: 모든 사용자) → 웹 앱 URL 복사
4. `index.html`의 제출 스크립트를 해당 URL로 JSON POST 하도록 수정 (현재는 구글 폼 방식이므로 코드 변경 필요)
5. 코드를 수정한 뒤에는 **배포 → 배포 관리 → 새 버전**으로 재배포해야 반영됩니다.
