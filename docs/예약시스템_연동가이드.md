# 예약 폼 → 구글 시트 + 이메일 알림 연동 가이드

초보자도 그대로 따라 하면 되는 순서입니다. 총 5단계, 10분 정도 걸립니다.

## 1단계. 구글 시트 새로 만들기
1. [sheets.new](https://sheets.new) 접속 (구글 계정 로그인 상태여야 함)
2. 시트 이름을 `김태효탑내과_예약요청`으로 변경

## 2단계. Apps Script 붙여넣기
1. 시트 상단 메뉴 **확장 프로그램 → Apps Script** 클릭
2. 열린 편집기의 기존 코드를 모두 지우고, 이 폴더의 `apps-script-code.gs` 파일 내용을 전부 복사해서 붙여넣기
3. 코드 2번째 줄의 `your-email@example.com`을 **알림받을 실제 이메일 주소**로 바꾸기 (예: 원장님이나 담당자 이메일)
4. 상단 저장 아이콘(💾) 클릭

## 3단계. 웹앱으로 배포하기
1. 편집기 오른쪽 위 **배포 → 새 배포** 클릭
2. 유형 선택(⚙️)에서 **웹 앱** 선택
3. 아래 설정:
   - 실행 사용자: **나**
   - 액세스 권한이 있는 사용자: **모든 사용자**
4. **배포** 클릭 → 구글 계정 권한 승인 (경고 화면이 나오면 "고급" → "이동" 클릭해서 승인)
5. 배포 완료 후 나오는 **웹 앱 URL**을 복사 (`https://script.google.com/macros/s/xxxx/exec` 형태)

## 4단계. 홈페이지 코드에 주소 붙여넣기
1. `website/index.html` 파일을 열기
2. 파일 맨 아래 `<script>` 안에서 아래 줄을 찾기:
   ```js
   const GAS_ENDPOINT = 'PUT_YOUR_APPS_SCRIPT_WEB_APP_URL_HERE';
   ```
3. `'PUT_YOUR_APPS_SCRIPT_WEB_APP_URL_HERE'` 부분을 3단계에서 복사한 웹 앱 URL로 교체

## 5단계. 테스트
1. 홈페이지를 브라우저에서 열고 예약 폼을 실제로 한 번 제출
2. 구글 시트에 새 행이 추가되는지 확인
3. 2단계에서 입력한 이메일로 알림 메일이 오는지 확인

## 참고
- 코드를 수정한 뒤에는 **배포 → 배포 관리 → 수정(연필 아이콘) → 새 버전 → 배포**를 해야 변경사항이 반영됩니다. URL은 그대로 유지됩니다.
- 완전히 무료이며, 구글 시트가 곧 예약 데이터베이스 역할을 합니다.
- 이메일 알림 주소는 원장님(또는 접수 담당자)이 실제로 확인하는 주소로 넣어야 실용적입니다.
