# 트래킹 코드 설치 가이드 (스텝바이스텝)

김태효탑내과의원 홈페이지에 설치하면 좋은 도구는 4가지입니다. 위에서부터 순서대로 하시면 되고, ①②③은 필수, ④는 광고를 시작할 때 하시면 됩니다.

| 순서 | 도구 | 용도 | 비용 |
|---|---|---|---|
| ① | 구글 서치콘솔 | 구글 검색에 사이트 등록·노출 확인 | 무료 |
| ② | 네이버 서치어드바이저 | 네이버 검색에 사이트 등록 (한국 병원은 필수!) | 무료 |
| ③ | 구글 애널리틱스(GA4) | 방문자 수·유입 경로·예약 버튼 클릭 분석 | 무료 |
| ④ | 네이버 프리미엄 로그분석 | 네이버 검색광고를 할 때 전환 측정 | 무료 |

> **코드를 붙여넣는 위치**: 모든 HTML 파일(index.html, about.html, checkup.html, endoscopy.html, ultrasound.html, wellcheck.html, pricing.html, privacy.html)의 `</head>` 바로 위에 안내 주석을 미리 넣어두었습니다.
> - `<!-- ▼ 사이트 소유확인 메타태그는 이 줄 아래에 붙여넣으세요 -->` ← ①②의 소유확인 메타태그
> - `<!-- ▼ GA4 등 분석 트래킹 코드는 이 줄 아래에 붙여넣으세요 -->` ← ③④의 스크립트
>
> 소유확인 메타태그는 **index.html에만 넣어도 되지만**, 분석 스크립트(③④)는 **8개 파일 모두**에 넣어야 전체 페이지가 측정됩니다.

---

## ① 구글 서치콘솔 (Google Search Console)

구글 검색 결과에 사이트가 나오게 등록하고, 어떤 검색어로 유입되는지 확인하는 도구입니다.

1. [search.google.com/search-console](https://search.google.com/search-console) 접속 → 구글 계정 로그인
2. "속성 추가" → 오른쪽의 **URL 접두어** 선택 → `https://kimtaehyo-clinic-website.vercel.app/` 입력
3. 소유 확인 방법 중 **HTML 태그** 선택 → 아래 같은 코드가 나옵니다:
   ```html
   <meta name="google-site-verification" content="abc123...">
   ```
4. 이 한 줄을 복사해서 **index.html**의 `<!-- ▼ 사이트 소유확인 메타태그... -->` 주석 바로 아래에 붙여넣기
5. 사이트를 배포(저장 후 git push)한 뒤, 서치콘솔로 돌아와 **확인** 버튼 클릭
6. 확인 완료 후 왼쪽 메뉴 **Sitemaps** → `sitemap.xml` 입력 → 제출
   (사이트맵은 이미 만들어져 있습니다: `/sitemap.xml`)

✅ 완료 확인: 며칠 뒤 서치콘솔 "실적" 메뉴에 노출/클릭 수가 잡히기 시작합니다.

## ② 네이버 서치어드바이저

한국 환자 대부분이 네이버에서 검색하므로 병원 사이트는 이게 가장 중요합니다.

1. [searchadvisor.naver.com](https://searchadvisor.naver.com) 접속 → 네이버 로그인 → 오른쪽 위 **웹마스터 도구**
2. 사이트 등록: `https://kimtaehyo-clinic-website.vercel.app` 입력
3. 소유 확인 방법 중 **HTML 태그** 선택 → 아래 같은 코드가 나옵니다:
   ```html
   <meta name="naver-site-verification" content="xyz789...">
   ```
4. **index.html**의 소유확인 주석 아래(①에서 넣은 구글 태그 다음 줄)에 붙여넣고 배포 → **소유확인** 클릭
5. 확인 완료 후 왼쪽 메뉴 **요청 → 사이트맵 제출** → `https://kimtaehyo-clinic-website.vercel.app/sitemap.xml` 제출
6. **요청 → 수집 요청**에서 대표 페이지 8개 URL을 각각 등록해두면 반영이 빨라집니다.

✅ 추가 팁: [네이버 스마트플레이스](https://smartplace.naver.com)에 병원을 등록/관리하고, 스마트플레이스 프로필의 "홈페이지" 항목에 이 사이트 주소를 넣어주세요. 지역 검색("진주 내과")에서는 스마트플레이스가 홈페이지보다 먼저 노출됩니다.

## ③ 구글 애널리틱스 GA4

하루 방문자 수, 어느 페이지를 많이 보는지, 예약 폼을 몇 명이 제출했는지 볼 수 있습니다.

1. [analytics.google.com](https://analytics.google.com) 접속 → 구글 계정 로그인
2. 관리(⚙️) → **계정 만들기** → 계정 이름: `김태효탑내과의원`
3. 속성 만들기 → 속성 이름: `홈페이지` / 시간대: 대한민국 / 통화: KRW
4. 플랫폼에서 **웹** 선택 → 사이트 주소 입력 → 만들면 **측정 ID**(`G-XXXXXXXXXX`)가 발급됩니다
5. 아래 코드에서 `G-XXXXXXXXXX` 두 곳을 본인 측정 ID로 바꾼 뒤, **8개 HTML 파일 모두**의 `<!-- ▼ GA4 등 분석 트래킹 코드... -->` 주석 아래에 붙여넣기:
   ```html
   <!-- Google tag (gtag.js) -->
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
   <script>
     window.dataLayer = window.dataLayer || [];
     function gtag(){dataLayer.push(arguments);}
     gtag('js', new Date());
     gtag('config', 'G-XXXXXXXXXX');
   </script>
   ```
6. (선택이지만 추천) **예약 요청 완료를 전환으로 측정하기**
   index.html 맨 아래 스크립트의 `showConfirm` 함수 안에 한 줄을 추가합니다:
   ```js
   function showConfirm(){
     form.style.display = 'none';
     document.getElementById('confirmBox').classList.add('show');
     showToast();
     if (typeof gtag === 'function') gtag('event', 'reservation_submit'); // ← 이 줄 추가
   }
   ```
   GA4 관리 → 이벤트에서 `reservation_submit`을 **주요 이벤트(전환)**로 표시하면, "이번 달 예약 요청 몇 건"을 바로 볼 수 있습니다.

✅ 완료 확인: GA4 왼쪽 메뉴 **보고서 → 실시간**을 열어놓고 휴대폰으로 사이트에 접속하면 방문자 1이 잡힙니다.

## ④ 네이버 프리미엄 로그분석 (검색광고 시작할 때)

네이버 검색광고(파워링크)를 집행하면 "광고 클릭 → 예약 요청"까지 추적할 수 있습니다. 광고를 하지 않는다면 건너뛰세요.

1. [searchad.naver.com](https://searchad.naver.com) → 광고 계정 생성
2. 도구 → **프리미엄 로그 분석** 신청 → 발급되는 공통 스크립트를 ③과 같은 위치(8개 파일 모두)에 붙여넣기
3. 전환 스크립트는 예약 완료 시점(위 `showConfirm` 함수)에 넣으면 됩니다.

## 자주 하는 실수 체크리스트

- [ ] 코드를 붙여넣고 **배포(git push)를 안 함** → 붙여넣기 후 반드시 배포까지 해야 인식됩니다.
- [ ] GA4 코드를 index.html에만 넣음 → 8개 파일 모두 넣어야 서브페이지 방문이 측정됩니다.
- [ ] 소유확인 메타태그를 지움 → 확인 후에도 **지우면 안 됩니다** (주기적으로 재확인함).
- [ ] `G-XXXXXXXXXX`를 실제 측정 ID로 안 바꿈.

## 나중에 실제 도메인(예: kimtaehyotop.co.kr)을 연결하면

vercel.app 주소 대신 병원 도메인을 쓰게 되면 아래를 한 번에 바꿔야 합니다:

1. 8개 HTML의 `canonical`, `og:url`, `og:image` 주소와 JSON-LD 안의 `url` (일괄 찾아바꾸기: `kimtaehyo-clinic-website.vercel.app` → 새 도메인)
2. `sitemap.xml`, `robots.txt`, `llms.txt` 안의 주소
3. 구글 서치콘솔·네이버 서치어드바이저에 **새 도메인으로 속성을 다시 등록**하고 사이트맵 재제출
4. 네이버 스마트플레이스의 홈페이지 주소 변경
